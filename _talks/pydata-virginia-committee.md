Member of the organizing committee for PyData Virginia, bringing the global PyData conference series to the Commonwealth. Working to build a regional conference focused on Python tools for data science, machine learning, and scientific computing.

**Role:** Organizing Committee Member (2023 - Present)

**Responsibilities:**
* Program development and speaker selection
* Community outreach and engagement
* Conference logistics and planning
* Building partnerships with local organizations

**Focus:** Creating a regional conference that brings together Python practitioners, data scientists, and researchers to share knowledge and advance the use of Python in data-intensive applications.

**About PyData:** PyData is an educational program of NumFOCUS, providing a forum for the international community of users and developers of data analysis tools to share ideas and learn from each other.
